# BuscaCentral.com.br

Central brasileira de ferramentas online gratuitas.
Consultas rápidas, sem cadastro e sem custo.

## 🌐 Acesse o site
[buscacentral.com.br](https://buscacentral.com.br)

## 🛠️ Ferramentas disponíveis

### 📋 Documentos
- Gerador de CPF
- Validador de CPF
- Gerador de CNPJ
- Validador de CNPJ
- Consulta CNPJ (Receita Federal)

### 📍 Localização
- Busca de CEP
- Busca reversa (rua → CEP)

### 💰 Financeiro
- Cotação de Moedas em tempo real
- Criptomoedas (Top 10)
- Tabela FIPE
- Juros Compostos
- Conversor CLT → PJ
- Calculadora de Financiamento de Carro
- Calculadora de Rescisão Trabalhista
- Simulador CDB / Tesouro Direto

### 🔧 Utilidades
- Gerador de QR Code
- Gerador de Senha
- Comparador de Textos
- Conversor de Imagens
- Seletor de Cores
- Extrator de Emails
- Link WhatsApp
- PIX Copia e Cola
- Formatador JSON/XML
- Calculadora de IMC
- Dias Úteis entre Datas

## 🚀 Stack

- HTML5 + CSS3 + JavaScript puro (zero framework)
- Hospedagem: Vercel
- Deploy automático via GitHub

## 🔌 APIs utilizadas

| API | Uso | Custo |
|---|---|---|
| ViaCEP | Busca de CEP | Gratuita |
| AwesomeAPI | Cotação de moedas | Gratuita |
| CNPJ.ws | Consulta CNPJ | Gratuita |
| parallelum.com.br | Tabela FIPE | Gratuita |

## 💡 Como rodar localmente

Clone o repositório e abra o index.html no navegador.
Não há dependências, build ou configuração necessária.

```bash
git clone https://github.com/buscacentral/busca-central.git
cd busca-central
# Abra o index.html no navegador
```
